Graphic Designer/User Interface Designer

Position: Graphic Designer/User Interface Designer
Department: IT
Position: Project Manager
Job Summary:
The Graphic Designer/ User Interface Builder will be responsible for designing an app interface that is both visually appealing and simple for the user to understand, with the screens being easy to navigate.
This is achieved by creating prototypes and presenting them to stakeholders and other developers working on the project for feedback. Being able to analyse customer feedback on the apps functionality and overall usability is a requirement for this application to be a success.
The app interface's quality can be determined by the App’s ratings and overall growth in downloads and subscriptions.
As we are a startup company, this will be a six-month contracted position, with ongoing maintenance work when required.
Please note: a contract and non-disclosure agreement will be required to ensure intellectual property is protected, and remains the property of PANNM.
Responsibilties and Duties:
The key duties and responsibilities for this role include, but are not limited to:
• Create prototypes for the app interface.
• Plan and implement new designs.
• Enhance the existing user interface.
• Test new ideas before applying them.
• Communicating with stakeholders and other developers.
• User research for app interface improvements.
Key Relationships:
Internal • Project Manager
• Front End Developer
• Back End Developer
• Marketing Manager
Key Decision Making in this Role:
• Deciding on app visual design (typography, buttons, colour palette).
• Deciding on app interaction (animation, interactivity, prototypes).
• Deciding on user feedback that is most beneficial to app design.
Key Chanllenges:
• Ensuring the product functions across all devices
• Conducting research under time constraints.
• Determining app features that can increase usability. (USABILITYGEEK 2019)
Qualifications and Experience:
Qualifications: • Tertiary degree in a design-related field (Preferred)
Experience: • Experience in Graphic Design/ Web Design/ UI Design
Knowledge/skills: • Interpersonal and communication skills
• Pride in own work
• Critical thinking
• Time-management skills
• Knowledge of design products (eg. Photoshop, Illustrator, Acrobat)
• Experience with HTML5 and CSS3
Personal Requirements: • Ability to work independently when required.
• Ability to work in a team.
• Strong work ethic and attention to detail.
• Sense of ownership and pride.
