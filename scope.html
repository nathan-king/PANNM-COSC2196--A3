Scope, Limits, Tools & Technologies

Scope & Limits

The project that we plan on completing will allow users to enter products, their expiry date and storage method, which will then be organised into a database using these parameters. The app will be notified when the expiration date comes near with alerts and ideas on using the product before it is unfit for consumption. Application Insights will define how much waste has been prevented over time and the overall success. The app will provide some information from an open-source database using a public API provided by Open Food Facts. The database will be used to present the user with the necessary information regarding their scanned products. However, we will not be using any of the provided nutritional information or product ingredients as this data doesnâ€™t fall into the initial aims of this project.
